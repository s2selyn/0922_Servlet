POST방식으로 요청 후 응답데이터 받아보기

		특징 :
		1. POST방식으로 요청을 보내면 BODY영역에 데이터를 포함시켜서 요청을 보냄 

		=> 사용자가 입력한 값(데이터)들이 URL에 노출되지 않음 

		=> 로그인, 회원 가입 같은 기능들은 POST가 적합함(보안이 필요한 내용) 

		
		2. 전송하는 길이에 제약이 없음 

		=> 즉, 게시글 작성 같은 경우 POST방식이 적합함 


		
		
		
		3. 최대 요청 받는 시간이 존재함 

		
		4. 보편적으로 새롭게 데이터를 만들어내는 기능에 사용됨
